Question-1: What are the different ways to select an element in the DOM?

select elements in the DOM using methods like getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML includes HTML tags, innerText renders visible text only, and textContent includes all text content (visible and hidden) without HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where you attach a single event listener to a parent element to handle events for its child elements, improving performance and efficiency.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event propagates from the target element up through its ancestors in the DOM hierarchy.
